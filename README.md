# Fake_News_Prediction
Machine Learning project for detecting fake news articles using Natural Language Processing (NLP), TF-IDF Vectorization, and Logistic Regression.
# Fake News Detection Using Machine Learning

## Project Overview

This project uses Machine Learning and Natural Language Processing (NLP) techniques to classify news articles as Real or Fake.

The model is trained on a dataset containing real and fake news articles. Text preprocessing techniques such as stopword removal, stemming, and TF-IDF vectorization are used before training a Logistic Regression classifier.

## Dataset

The project uses:

* True.csv (Real News Articles)
* Fake.csv (Fake News Articles)
